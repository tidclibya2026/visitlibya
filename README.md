# Visit Libya | زور ليبيا — Frontend Prototype

نسخة أولية احترافية للواجهة الأمامية لمنصة Visit Libya. مبني بـ HTML + CSS + JavaScript فقط (بدون React أو Backend).

ملفات مضافة:
- `index.html` — الصفحة الرئيسية (RTL افتراضي).
- `style.css` — أنماط وتصميم الهوية البصرية.
- `main.js` — جميع التفاعلات: قائمة الموبايل، الأكورديون، الـ AI chat التجريبي، والـ reveal on scroll.

الصور:
- استخدم المجلد `imges/` لوضع صور الوجهات. التصميم يستخدم مسارات نسبية مثل `imges/1.jpg` إلخ.
- إذا لم توجد صورة، يعرض الكرت لونا تراثيا ولا يكسر التصميم.

تشغيل محلي:
1. افتح المجلد في متصفحك (فتح `index.html`).
2. لا حاجة لخادم أو باك-إند.

ملاحظات:
- التصميم Responsive ويتعامل مع RTL وLTR عبر زر اللغة التجريبي.
- قسم VisitLibyaAI هو واجهة محلية تجريبية بدون باك-إند وترد بردود مبسطة بكلمات مفتاحية.

إذا رغبت، أستطيع:
- إضافة ملفات صور افتراضية أو أيقونات SVG.
- تحسين تجربة اللغة الإنجليزية ونسخ المحتوى بالإنجليزية.
- إنشاء نسخة جاهزة للطباعة أو PDF للعرض الرسمي.
